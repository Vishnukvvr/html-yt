HTML

HTML stands for Hypertext Markup Language.

It is the standard language used to create and structure web pages.

HTML provides the structure for web content, such as text, images, links, forms, and multimedia.

HTML uses tags to define elements, typically in the form of content.

Opening tag:-<>

Closing tag:-</>


HTML Structure:-

            
                <!DOCTYPE html>
                <html>
                <head>
                    <title>My Web Page</title>
                </head>
                <body>
                    <h1>Hello, World!</h1>
                    <p>This is a paragraph.</p>
                </body>
                </html>

1:- <!DOCTYPE html>
The <!DOCTYPE html> declaration is used to specify that the document is an HTML5 document, ensuring the browser renders it using the HTML5 standard.

2:-<html> Element:
The <html> tag serves as the root element of an HTML page, enclosing all the content and other elements of the page.

3:-<head> Element:
The <head> element contains metadata and other information about the document, such as the character set, links to stylesheets, and JavaScript files, as well as the page's title.

4:-<title> Element:
The <title> element sets the title of the HTML document, which is displayed in the browser's title bar or tab, helping users identify the page.

5:-<body> Element:
The <body> tag defines the main content of the webpage. It holds all the visible elements such as text, images, videos, tables, links, and other interactive content.
